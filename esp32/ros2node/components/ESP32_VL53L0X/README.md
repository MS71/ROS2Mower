# ESP32_VL53L0X

VL53L0X driver as ESP-IDF component

## Example

```sh
git clone https://github.com/kerikun11/ESP32_VL53L0X.git
cd ESP32_VL53L0X/example
make defconfig flash monitor
```

## Dependency

- [ESP-IDF](https://github.com/espressif/esp-idf)
